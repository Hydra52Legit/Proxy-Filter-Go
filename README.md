# 🚀 HTTP Proxy Filter

HTTP прокси-сервер с фильтрацией трафика на основе черного списка доменов.

## ✨ Возможности

- 🔄 **Проксирование HTTP трафика** — перенаправляет запросы к целевым серверам
- 🚫 **Фильтрация доменов** — блокировка доступа к сайтам из черного списка
- 📊 **Статистика в реальном времени** — веб-интерфейс с метриками
- 🧩 **Поддержка поддоменов** — автоматическая блокировка всех поддоменов
- 💾 **Гибкое конфигурирование** — черный список загружается из текстового файла

## 🏗️ Архитектура

Проект построен на принципах **композиции** (вместо наследования):

- `Config` — загрузка и хранение конфигурации
- `Filter` — бизнес-логика проверки доменов
- `Server` — HTTP сервер и проксирование
